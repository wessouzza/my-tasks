# Nome do Projeto Fullstack

Este é um projeto fullstack desenvolvido com Java, Spring Boot, React, PostgreSQL e Swagger. A aplicação é uma Todo List, criada especificamente com o intuito de demonstrar minhas habilidades técnicas para uma vaga.
